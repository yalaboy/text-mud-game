# 텍스트 기반 MUD 게임

간단한 텍스트 기반 MUD 게임입니다.

## 특징
- 중앙 마을, 동쪽 초보자 사냥터, 서쪽 중급자 던전
- 직업 선택: warrior, archer, mage
- 레벨업 시 스탯 포인트 5개 획득
- 직업별 스킬과 장비 조건
- 이동, 탐험, 상태 확인, 스탯 투자, 장비 장착 지원

## 실행 방법
```bash
python3 main.py
```

## 기본 명령어
- look: 지도와 현재 위치 확인
- go east / go west: 이동
- status: 상태 확인
- train: 스탯 투자
- skill: 스킬 목록 확인
- equip: 장비 장착 시도
- explore: 현재 지역 탐험
- quit: 종료
